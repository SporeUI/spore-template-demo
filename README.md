# spore-example

spore 引擎项目示例

## 配置

```bash
# 安装项目管理工具 spore
npm i -g @spore-ui/spore
# 设置项目模板
spore template set example git@github.com:SporeUI/spore-template-demo.git
```

## 创建项目

```bash
# 创建项目到 app 目录
spore create -t example
# 创建项目到本目录
spore create -t example .
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```
